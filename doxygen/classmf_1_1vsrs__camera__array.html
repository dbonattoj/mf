Detailed Description

Reader of camera array format used by VSRS.

File contains array of names projection cameras, defined using their extrinsic and intrinsic matrices, in plain text format. Some camera array files contain additional fourth 0 0 0 1 row of extrinsic matrix, while some omit it. fourth_extrinsic_row must be set accordingly.
